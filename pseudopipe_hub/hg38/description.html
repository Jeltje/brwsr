Description

PseudoPipe, automated pseudogene identification

Special note: Track is searchable by parent Hugo and Ensembl (ENSG) IDs. This allows you to get an estimate of the number of pseudogenes for a particular parent.

Display Conventions

By default pseudogenes are labeled by parent Hugo ID and type of pseudogene. The Hugo IDs are based on the Ensembl gene IDs provided by the Gerstein lab after dataset creation and not all PseudoPipe IDs could be mapped back to their original Ensembl IDs. In these cases the gene ID is listed as NA.
Pseudogenes are colored by type.

  • orange - unspecified pseudogene
  • blue - processed pseudogene
  • olive green - unprocessed pseudogene

Mouseover on items shows the PseudoPipe ID and Ensembl gene name. Clicking on any item brings up a table with other identifiers.

Data Access

The raw data can be explored interactively with the Table Browser or the Data Integrator. For automated analysis, the data may be queried from our REST API.
Please refer to our mailing list archives for questions, or our Data Access FAQ for more information.

Methods

Annotations were filtered to remove duplicates and non standard chromosomes, and overlapping exons merged if they belonged to the same transcript.
